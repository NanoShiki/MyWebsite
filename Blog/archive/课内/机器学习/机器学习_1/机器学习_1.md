# 机器学习 - 第1课

## 一、机器学习 vs 模式识别

### 名称由来

- 课程叫"机器学习"而不是"模式识别"
- 本质上没有区别，只是侧重点不同

### 两者的本质

| 概念 | 说明 |
|------|------|
| **模式识别** | 从数据中识别或发现规律并加以运用 |
| **机器学习** | 让机器能够学习，学习规律并用于预测 |

**核心：两者本质一样，只是学科划分和学术背景不同导致名称差异。**

---

## 二、规律识别的例子

### 数列找规律

```
1, 1, 2, 3, 5, 8, 13, ?
```

- **斐波那契数列**
- 规律：前两个数相加 = 第三个数
- **完全确定、完全正确**

### 有噪声的数据

```
1, 1, 2.1, 3.2, 5.3, 8.1, ?
```

- 接近但不完全符合
- **机器学习：大概率符合即可，不需要完全确定**

## 三、机器学习的工作流程

### 类比：待定系数法求函数

高中数学的例子：

1. **收集数据** → 给出三个点
2. **建模** → 假设是抛物线：`Y = AX² + BX + C`
3. **训练** → 列方程组求解参数 A、B、C
4. **测试** → 给定 X，预测 Y

### 机器学习流程

| 步骤 | 说明 |
|------|------|
| **收集数据** | 有数据才能发现规律 |
| **建模** | 构建模型架构（函数形式） |
| **训练** | 确定模型参数 |
| **测试** | 用模型进行预测 |

### 模型架构的选择

给定三个点，不一定是抛物线：
- 可以是 sin 曲线
- 可以是指数函数
- 可以是多项式
- **函数形式（模型架构）可以灵活变化**

## 四、机器学习的三种范式

### 1. 监督学习（Supervised Learning）

**特点：问题 + 答案**

- 老师告诉你这个答案是对的
- 按这个答案学习
- **输入 X，输出 Y（标签）**

**例子：**
- 分类：这张图是猫还是狗？
- 回归：预测房价
- 翻译：中文 → 英文

### 2. 无监督学习（Unsupervised Learning）

**特点：只有数据，没有答案**

- 只有 X，没有 Y
- 让机器自己发现规律

**例子：**
- 聚类：把相似的数据分组
- 降维：PCA、t-SNE
- 自编码器：自己生成数据

### 3. 强化学习（Reinforcement Learning）

**特点：问题 + 答案 + 评价（奖励）**

- 不仅给出答案，还要评价答案好坏
- 通过奖励反馈学习
- 评价之上可能还有评价

**类比：**
- 监督学习：老师告诉你这个答案是对的
- 强化学习：不仅要答案，还要看整个对话过程，不断评价

## 五、强化学习的应用案例

### 人类反馈强化学习（RLHF）

现在的 AI 对话系统：
- 一开始回答可能不好
- 用户反馈：点赞、点踩、反驳
- AI 通过这些反馈学习改进
- **通过数据操纵 AI 的行为**

### 机器人抓取杯子

**监督学习：**
- 不停采集数据，操控机器人

**强化学习：**
- 让机器人自己尝试
- 成功给奖励，失败给惩罚
- 自己学会抓取

## 六、数据与特征

### 数据集的组成

| 概念 | 说明 |
|------|------|
| **对象** | 数据样本（一行） |
| **属性** | 特征（一列） |
| **属性值** | 特征的取值 |

### 属性类型

| 类型 | 说明 | 例子 |
|------|------|------|
| **离散属性** | 有限取值 | 性别（男/女） |
| **二元属性** | 是/非 | 是否及格 |
| **连续属性** | 数值范围 | 身高、体重 |

### 数据集的特性

| 特性 | 说明 |
|------|------|
| **维度** | 属性数量（列数），维度越多越复杂 |
| **维数灾难** | 维度高但数据量少 |
| **稀疏性** | 大部分属性值为空 |
| **分辨率** | 数据精度，越高越难学习 |

## 七、数据表示

### 数据矩阵

- M 行 N 列的矩阵
- 行：对象/样本
- 列：属性/特征

### 图像表示

**灰度图像：**
- 28 × 28 = 784 个像素
- 单通道

**RGB 彩色图像：**
- 28 × 28 × 3
- R、G、B 三个通道

**多张图像：**
- 28 × 28 × 3 × 图片数量

## 八、机器学习应用场景

### 监督学习应用

| 任务 | 输入 | 输出 |
|------|------|------|
| **分类** | 图片 | 类别（猫/狗） |
| **回归** | 房屋信息 | 价格 |
| **目标检测** | 图片 | 边界框坐标 |
| **语音识别** | 音频 | 文字 |
| **机器翻译** | 中文 | 英文 |
| **对话系统** | 用户问题 | AI 回答 |

### 无监督学习应用

- **聚类**：自动分组
- **降维**：减少维度
- **自编码器**：自己生成数据
- **去噪**：去除噪声
