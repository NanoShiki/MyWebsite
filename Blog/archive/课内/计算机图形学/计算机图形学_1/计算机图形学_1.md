# 计算机图形学 - 第1课

## 一、课程介绍

### 课程定位

- **课程性质**：计算机图形学入门课程
- **所属学院**：信息学院
- **课程特殊性**：增加了 1:1 的上机实验课

### 课程要求

- 大纲看起来要求高，但实际要求**不算太高**
- 考虑到大家学业繁忙
- 强调"硬核度"但实际适中

---

## 二、学科背景与方向

### 学科方向

| 方向 | 说明 |
|------|------|
| **视觉** | 计算机视觉、图像处理 |
| **图形学** | 渲染、几何、动画 |
| **智能视觉** | AI + 视觉 |
| **物理仿真** | 物理引擎、游戏 |

### 研究现状

- **纯粹的图形学研究**：非常难（渲染、几何都很难）
- **大多数人**：做可视化、视觉等交叉方向
- **背景**：很多博导教授来自 CAD 实验室

---

## 三、课程历史与对比

### 研究生课程对比

| 类型 | 说明 |
|------|------|
| **本科生课** | 本课程 |
| **研究生课** | 更高要求，一个月左右做一个项目 |

### 研究生项目要求

- **从底层写起**：不用公开的 3D 库
- **自己实现**：相机、渲染管线
- **从零开始**：无任何功能评价标准

### 实际情况

- 班上几十人，只有 **3-4 人**能真正做出来
- 大多数人：看别人代码，看懂后自己重新写
- 有人：独立从头写（对性格要求高）
