# 🌟 myWeb 项目

## ⚠️ 项目声明

**本项目为纯AI项目，旨在通过增加AI使用率来学习AI。**

---

myWeb 是一个个人项目集合仓库，包含了独立的博客子项目。

```

│   ├── post.html       # 📖 文章详情页
│   ├── config.json     # ⚙️ 博客配置
│   ├── update_config.py # 🛠️ 配置更新脚本
│   └── 使用说明.md      # 📘 脚本使用说明
├── wiki/               # 📖 技术文档
```

## 🛠️ 技术栈

### 整体
- **版本控制**: Git 📦
- **操作系统**: macOS 🍎

### Blog
- **静态页面**: HTML 🌐
- **配置文件**: JSON ⚙️
- **文章格式**: Markdown 📝
- **代码高亮**: highlight.js 🎨
- **Markdown 解析**: marked.js ✨
- **数学公式**: KaTeX 🧮

## 📝 Blog 功能

- ✅ 文章归档与分类
- ✅ 配置化管理
- ✅ 多级分类导航与面包屑
- ✅ 目录导航（TOC）
- ✅ 代码高亮（支持 40+ 语言）
- ✅ 一键复制代码
- ✅ 图片放大查看
- ✅ 响应式设计
- ✅ 数学公式渲染（KaTeX）
- ✅ 文章标题从Markdown读取
