# 🌟 myWeb 项目

myWeb 是一个个人项目集合仓库，包含了独立的博客子项目。

## 📂 目录结构

```
myWeb/
├── Blog/               # 📝 个人博客
│   ├── archive/        # 📚 博客文章归档
│   ├── index.html      # 🏠 博客首页
│   ├── post.html       # 📖 文章详情页
│   ├── config.json     # ⚙️ 博客配置
│   └── update_config.py # 🛠️ 配置更新脚本
├── wiki/               # 📖 技术文档
│   ├── readme.md       # 📄 本文件
│   ├── blog.md         # 📝 Blog 项目文档
│   ├── 部署指南.md     # 🚀 部署指南
│   └── teach/          # 🎓 教学文档
├── index.html          # 🏠 个人主页
├── readme.md           # 📄 本文件
└── .gitignore
```

## 🛠️ 技术栈

### 整体
- **版本控制**: Git 📦
- **操作系统**: macOS 🍎

### Blog
- **静态页面**: HTML 🌐
- **配置文件**: JSON ⚙️
- **文章格式**: Markdown 📝
- **代码高亮**: highlight.js 🎨
- **Markdown 解析**: marked.js ✨

## 📝 Blog 功能

- ✅ 文章归档与分类
- ✅ 配置化管理
- ✅ 目录导航（TOC）
- ✅ 代码高亮（支持 40+ 语言）
- ✅ 一键复制代码
- ✅ 图片放大查看
- ✅ 响应式设计
