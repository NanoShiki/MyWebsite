## 7.2 实战一：添加新博客文章

### 步骤

```bash
# 1. 进入 blog 目录
cd blog

# 2. 在 archive 下创建新文章文件夹
mkdir -p archive/我的分类/我的新文章

# 3. 创建 Markdown 文件
touch archive/我的分类/我的新文章/我的新文章.md

# 4. 编辑文章内容（使用你喜欢的编辑器）
# 添加内容到 archive/我的分类/我的新文章/我的新文章.md

# 5. 运行更新脚本
python3 update_config.py

# 6. 查看更新后的 config.json
cat config.json

# 7. 提交更改
git add archive/我的分类/ config.json
git commit -m "添加新文章：我的新文章"

# 8. 推送 blog 子模块
git push

# 9. 返回主仓库，更新子模块引用
cd ..
git add blog
git commit -m "更新博客子模块"
git push
```

## 7.4 实战三：修改 Flask API

### 添加新的 API 端点

```python
# 编辑 ZmdDmg/backend/app.py

from flask import Flask, jsonify

app = Flask(__name__)

# 添加新的 API
@app.route('/api/hello', methods=['GET'])
def hello():
    return jsonify({
        "message": "Hello from Flask!",
        "status": "success"
    })

if __name__ == '__main__':
    app.run(debug=True, port=5000)
```

### 测试新 API

```bash
# 使用 curl 测试
curl http://localhost:5000/api/hello

# 或使用 Python
import requests
response = requests.get('http://localhost:5000/api/hello')
print(response.json())
```

## 7.5 实战四：创建 React 组件

### 新建组件

```jsx
// ZmdDmg/frontend/src/components/NewComponent.jsx
import { useState } from 'react';
import './NewComponent.css';

function NewComponent({ title }) {
    const [count, setCount] = useState(0);

    return (
        <div className="new-component">
            <h2>{title}</h2>
            <p>计数: {count}</p>
            <button onClick={() => setCount(count + 1)}>
                增加
            </button>
        </div>
    );
}

export default NewComponent;
```

### 添加样式

```css
/* ZmdDmg/frontend/src/components/NewComponent.css */
.new-component {
    background: white;
    padding: 20px;
    border-radius: 8px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

.new-component button {
    background: #667eea;
    color: white;
    border: none;
    padding: 10px 20px;
    border-radius: 4px;
    cursor: pointer;
}

.new-component button:hover {
    background: #5a6fd6;
}
```

### 在 App 中使用

```jsx
// ZmdDmg/frontend/src/App.jsx

function App() {
    return (
        <div>
            <NewComponent title="我的新组件" />
        </div>
    );
}
```

## 7.8 Git 工作流程

### 日常开发

```bash
# 1. 拉取最新代码
git pull
git submodule update --init --recursive

# 2. 创建功能分支
git checkout -b feature/my-feature

# 3. 开发...

```

## 7.9 常见问题解决

### 前端无法连接后端？

检查：
1. 后端是否启动？
2. 端口是否正确？
3. CORS 是否配置？

### 子模块找不到？

```bash
git submodule update --init --recursive
```

### 端口被占用？

```bash
# 查找占用端口的进程
lsof -ti:5000 | xargs kill -9
lsof -ti:5173 | xargs kill -9
lsof -ti:8000 | xargs kill -9
```

### npm install 失败？

```bash
# 删除 node_modules 重新安装
rm -rf node_modules package-lock.json
npm install
```
