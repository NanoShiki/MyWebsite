## 前置知识

- C++ 基础（你已经掌握！）
- UE 开发经验（可选，有助于理解）
- Git 基础

## 学习方式

1. **按顺序阅读** - 从第 1 章开始，循序渐进
2. **动手实践** - 边看边敲代码，不要只看不练
3. **参考项目** - 结合实际项目代码理解
4. **遇到问题** - 善用搜索引擎和文档

## 技术栈速览

```
myWeb 项目
├── 后端
│   ├── Python
│   └── Flask
├── 前端
│   ├── JavaScript
│   ├── React
│   ├── HTML
│   └── CSS
└── 版本控制
    └── Git (子模块)
```

## C++/UE 概念对应

| C++/UE | Web |
|--------|-----|
| 变量/函数 | 变量/函数 |
| 类/对象 | 类/对象 |
| Actor | 组件 (Component) |
| UPROPERTY | Props/State |
| UFUNCTION | 函数 |
| BeginPlay | useEffect (挂载) |
| Tick | requestAnimationFrame |
| Widget Tree | DOM Tree |
| 样式 | CSS |
