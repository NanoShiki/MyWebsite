# 第 5 章：前端基础（HTML + CSS）

## 前言

HTML 定义页面结构，CSS 定义页面样式。对于 UE 程序员，可以把 HTML 想象成 UMG 的 Widget 树，CSS 想象成 Widget 的样式。

## 5.1 HTML 基础

### HTML 文档结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
    <style>
        /* CSS 样式 */
    </style>
</head>
<body>
    <!-- 页面内容 -->
    <h1>标题</h1>
    <p>段落</p>
</body>
</html>
```

### 常用 HTML 标签

```html
<!-- 标题 -->
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>

<!-- 段落和文本 -->
<p>这是一个段落</p>
<strong>粗体</strong>
<em>斜体</em>
<br>  <!-- 换行 -->
<hr>  <!-- 分割线 -->

<!-- 链接 -->
<a href="https://example.com">链接</a>

<!-- 图片 -->
<img src="image.jpg" alt="图片描述">

<!-- 列表 -->
<ul>
    <li>无序列表项1</li>
    <li>无序列表项2</li>
</ul>

<ol>
    <li>有序列表项1</li>
    <li>有序列表项2</li>
</ol>

<!-- 容器 -->
<div>块级容器</div>
<span>行内容器</span>

<!-- 表单 -->
<input type="text" placeholder="文本输入">
<input type="number" placeholder="数字输入">
<button>按钮</button>

<!-- 语义化标签 -->
<header>页眉</header>
<nav>导航</nav>
<main>主要内容</main>
<article>文章</article>
<section>章节</section>
<aside>侧边栏</aside>
<footer>页脚</footer>
```

### div 和 class

```html
<!-- 使用 class 标记元素（类似 UE 的 Tag） -->
<div class="container">
    <div class="card">
        <h2 class="title">标题</h2>
        <p class="content">内容</p>
    </div>
</div>

<!-- 多个 class -->
<div class="card active">同时有两个 class</div>

<!-- 使用 id（唯一标识） -->
<div id="main-content">唯一元素</div>
```

## 5.2 CSS 基础

### 三种引入方式

```html
<!-- 1. 内联样式（不推荐） -->
<div style="color: red; font-size: 16px;">内容</div>

<!-- 2. 内部样式 -->
<head>
    <style>
        .title {
            color: blue;
        }
    </style>
</head>

<!-- 3. 外部样式（推荐） -->
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

### CSS 选择器

```css
/* 元素选择器 */
div {
    color: red;
}

/* class 选择器（最常用） */
.card {
    background: white;
}

/* id 选择器 */
#header {
    font-size: 24px;
}

/* 后代选择器 */
.container .card {
    margin: 10px;
}

/* 子选择器 */
.container > .card {
    padding: 20px;
}

/* 同时选择多个 */
h1, h2, h3 {
    font-weight: bold;
}

/* 伪类 */
button:hover {
    background: #eee;
}

button:active {
    background: #ddd;
}

input:focus {
    outline: 2px solid blue;
}
```

### 盒模型（Box Model）

```css
.box {
    /* 内容大小 */
    width: 200px;
    height: 100px;

    /* 内边距（内容和边框之间） */
    padding: 20px;
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;

    /* 边框 */
    border: 2px solid #333;
    border-radius: 8px;  /* 圆角 */

    /* 外边距（和其他元素之间） */
    margin: 20px;
    margin: 10px 20px;  /* 上下 10px，左右 20px */
}
```

### Flexbox 布局（最常用）

```css
.container {
    display: flex;

    /* 主轴方向 */
    flex-direction: row;          /* 水平（默认） */
    flex-direction: column;       /* 垂直 */
    flex-direction: row-reverse;  /* 水平反向 */

    /* 主轴对齐 */
    justify-content: flex-start;    /* 左对齐 */
    justify-content: center;        /* 居中 */
    justify-content: flex-end;      /* 右对齐 */
    justify-content: space-between;  /* 两端对齐 */
    justify-content: space-around;   /* 均匀分布 */

    /* 交叉轴对齐 */
    align-items: stretch;       /* 拉伸（默认） */
    align-items: flex-start;    /* 顶部对齐 */
    align-items: center;        /* 居中 */
    align-items: flex-end;      /* 底部对齐 */

    /* 换行 */
    flex-wrap: nowrap;  /* 不换行（默认） */
    flex-wrap: wrap;    /* 换行 */

    /* 间距 */
    gap: 20px;  /* 元素之间的间距 */
}

.item {
    /* 子元素 */
    flex: 1;  /* 占据剩余空间 */
    flex-grow: 1;   /* 放大比例 */
    flex-shrink: 1; /* 缩小比例 */
    flex-basis: 100px; /* 基础大小 */
}
```

Flexbox 示例：

```css
/* 水平居中 */
.center-horizontal {
    display: flex;
    justify-content: center;
}

/* 垂直居中 */
.center-vertical {
    display: flex;
    align-items: center;
    height: 100vh;
}

/* 完全居中 */
.center-both {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

/* 卡片网格 */
.card-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 1 1 250px;  /* 最小 250px，可伸缩 */
}
```

### Grid 布局（二维布局）

```css
.container {
    display: grid;

    /* 列 */
    grid-template-columns: 1fr 1fr 1fr;  /* 3 列等宽 */
    grid-template-columns: repeat(3, 1fr); /* 同上 */
    grid-template-columns: 200px 1fr 200px; /* 固定宽度 */

    /* 行 */
    grid-template-rows: auto;
    grid-template-rows: 100px 200px;

    /* 间距 */
    gap: 20px;
    row-gap: 10px;
    column-gap: 20px;
}
```

### 常用样式属性

```css
.element {
    /* 颜色 */
    color: #333;                   /* 文字颜色 */
    background-color: #f5f5f5;    /* 背景颜色 */
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

    /* 字体 */
    font-family: -apple-system, BlinkMacSystemFont, sans-serif;
    font-size: 16px;
    font-weight: 400;  /* 100-900，400 正常，700 粗体 */
    line-height: 1.6;  /* 行高 */
    text-align: center; /* left, center, right */

    /* 阴影 */
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
    text-shadow: 1px 1px 2px rgba(0, 0, 0, 0.1);

    /* 变换 */
    transform: translateY(-3px);  /* 上移 3px */
    transform: scale(1.05);       /* 放大 5% */
    transform: rotate(5deg);       /* 旋转 5 度 */

    /* 过渡动画 */
    transition: all 0.3s ease;
    transition: transform 0.2s, box-shadow 0.2s;

    /* 鼠标 */
    cursor: pointer;  /* 手型 */
    cursor: default;  /* 默认 */
    cursor: not-allowed; /* 禁止 */

    /* 显示 */
    display: block;    /* 块级 */
    display: inline;   /* 行内 */
    display: none;     /* 隐藏 */
    visibility: hidden; /* 隐藏但占位 */

    /* 位置 */
    position: relative; /* 相对定位 */
    position: absolute; /* 绝对定位 */
    position: fixed;    /* 固定定位 */
    position: sticky;   /* 粘性定位 */

    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    z-index: 10;  /* 层级 */
}
```

## 5.3 响应式设计

```css
/* 默认样式（移动端） */
.container {
    padding: 10px;
}

/* 平板 */
@media (min-width: 768px) {
    .container {
        padding: 20px;
    }
}

/* 桌面 */
@media (min-width: 1024px) {
    .container {
        padding: 40px;
        max-width: 1200px;
        margin: 0 auto;
    }
}
```

## 5.5 CSS 最佳实践

1. **使用 class 而不是 id**
2. **使用 Flexbox/Grid 布局**
3. **使用相对单位（rem, em, %）**
4. **保持选择器简单**
5. **使用 CSS 变量**

```css
/* CSS 变量 */
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --text-color: #333;
    --background-color: #f5f5f5;
    --shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
}

/* 使用变量 */
.button {
    background: var(--primary-color);
    box-shadow: var(--shadow);
}
```
