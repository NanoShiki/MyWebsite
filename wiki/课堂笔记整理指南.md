# 课堂录音转笔记提示词

## 笔记整理要求

### 核心原则
将课堂录音文本整理为结构化笔记时，必须遵循以下要求：

1. **字数控制**：总字数控制在 **1000字以内**
2. **内容完整**：确保涵盖课堂所有核心知识要点
3. **易于理解**：使用教学性、描述性语言，让读者能够快速轻松地理解吸收

---

## 整理步骤与格式

### 1. 标题与结构
- 使用清晰的标题层级
- 每个知识点用小标题分隔
- 使用列表、表格等形式提高可读性

### 2. 内容处理
- 提取核心概念，去除口语化重复
- 对复杂内容做适当补充解释
- 使用类比和例子帮助理解

### 3. 格式要求
```markdown
# [课程名称] - 第X课

## 一、[主要知识点1]
- 要点1
- 要点2

## 二、[主要知识点2]
- 要点1
- 要点2

## 小结
- 本课核心要点回顾
- 后续内容预告
```

---

## 关键提示词（复制使用）

```
请将以下课堂录音文本整理为课堂笔记，要求：
1. 字数控制在1000字以内
2. 确保内容完整，涵盖所有核心知识点
3. 使用教学性语言，添加适当解释，让读者能够快速轻松理解
4. 使用清晰的标题层级和列表格式

录音文本：
[在此粘贴录音文本]
```

---

## 质量检查清单

整理完成后，请检查：
- [ ] 字数在1000字以内
- [ ] 所有核心知识点都已涵盖
- [ ] 语言清晰易懂，无口语化混乱
- [ ] 结构清晰，便于快速查阅
- [ ] 对复杂内容有适当补充说明
