# 首页素材需求

首页场景图的**完整规范与完整提示词**，统一以 `wiki/站点前端素材需求.md` 为准。本文只保留首页条目的快速摘要，方便你在处理首页资源时快速查看。

## 对应文件

首页目前同时支持 **Hero 首屏视频** 与 **章节场景图**。统一优先放到：

```text
public/site-assets/home/hero/videos/
public/site-assets/home/scenes/
```

本轮正式接入文件名：

- `public/site-assets/home/hero/videos/manifest.json`
- `public/site-assets/home/hero/videos/ryu.mp4`
- `public/site-assets/home/scenes/camp-scene-day.jpg`
- `public/site-assets/home/scenes/camp-scene-night.jpg`
- `public/site-assets/home/scenes/trail-scene.jpg`

兼容旧路径：

```text
public/home-assets/
```

旧目录中的同名 `png / jpg / jpeg / webp` 仍然可以被首页识别，但后续新增资源建议只放到 `public/site-assets/home/` 这一套正式目录。

## Hero `mp4` 摘要

- 目录：`public/site-assets/home/hero/videos/`
- 清单：`public/site-assets/home/hero/videos/manifest.json`
- 当前接入：`ryu.mp4`
- 行为：首页一进入时优先显示 `mp4`；只在 Hero 首屏播放，离开 Hero 后淡出并暂停，回到 Hero 会重播
- 轮换：每次进入首页时，从 manifest 中随机挑一个可用 `mp4`
- 建议：只投喂 `mp4`，不要把 `gif` 当首页正式首屏背景

## 统一要求摘要

- 必须是**二次元图片**
- 必须是**动漫风格**
- 必须优先落在**日式西幻 / 中世纪奇幻 / 剑与魔法 / 龙与勇士**题材
- 必须**清新、干净、通透**
- 即使是黄昏、火光、余烬，也不能脏、旧污、阴森
- 不要人物、文字、Logo、UI、水印

## `camp-scene-day.*` / `camp-scene-night.*`

- 对应页面位置：首页第二屏 `营地`
- 推荐尺寸：`1536x1024`
- 当前命名：白昼版 `camp-scene-day.*`，夜晚版 `camp-scene-night.*`
- 兼容说明：若暂时只有一张营地图，也可继续沿用旧命名 `camp-scene.*` 作为通用 fallback
- 短版 prompt 摘要：二次元动漫风格、水彩感的日式西幻、中世纪奇幻营地场景插画，题材属于剑与魔法、龙与勇士世界，营地搭建在巨大白色遗迹附近，远处可见白色石材与金色装饰的高耸纪念塔 / 圣印塔；白昼版偏暖金黄昏、整备出发感；夜晚版允许烛火与夜幕，但仍必须清新干净、柔和通透，不能脏、旧污、阴森，中央区域留给内容卡，不要人物、文字、UI、Logo、水印，不要脏乱求生营地感

## `trail-scene.*`

- 对应页面位置：首页第三屏 `旅者足迹`
- 推荐尺寸：`1536x1024`
- 短版 prompt 摘要：二次元动漫风格、水彩感的日式西幻、中世纪奇幻旅途场景插画，题材属于剑与魔法、龙与勇士世界，道路可通往巨大白色遗迹与高耸纪念塔 / 圣印塔，白色石材、金色装饰、蓝天与云层、柔和而更热一点的火光、余烬、发光石径或足迹、中央留白，不要人物、文字、UI、Logo、水印，不要脏旧废墟和恐怖感

## 接入后的行为

- 没有图：首页继续使用纯 CSS 场景层
- 有图：切到对应章节时，图片与当前渐变、光影和环境层混合显示
- 图片不会直接盖住文字，而是作为章节背景增强层使用
